HelloKii for Node.js

HelloKii is a simple application which demonstrates common tasks you will likely need in any Kii Cloud application.

  • Signing in a new user
  • Logging in as an existing user
  • Saving data to an object in a bucket.

Other tutorials will walk you through more complex tasks in more detail. This tutorial will help you get started.

Setup

If you downloaded this tutorial via the developer portal, it will have automatically created an application named 'Kii Tutorial'.

The configuration in 'lib/modules/init.js'will have been edited with the app id, key and site inserted for you also.

  Kii.initializeWithSite(
    "__KII_APP_ID__",
    "__KII_APP_KEY__",
    __KII_APP_SITE__
  );
